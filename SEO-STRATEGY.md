## On-Page SEO Rules

### Title Tags
- Include primary keyword at the start
- Add year for freshness ("2025")
- Max 60 characters
- Include power word ("Guide", "Best", "How to")

### Meta Descriptions
- Include primary keyword naturally
- Include a call-to-action
- Max 155 characters
- Answer the search intent directly

### Content Structure
- H1: Page title (include primary keyword)
- H2: Main sections (include related keywords)
- H3: Sub-sections (use long-tail variations)
- First paragraph: Include primary keyword within first 100 words
- Include FAQ section for rich snippet eligibility

### Internal Linking
- Every guide links to 3+ related guides
- Use descriptive anchor text (not "click here")
- Link from pillar pages to cluster articles
- Update old articles when publishing new ones

### Image SEO
- All images need descriptive alt text
- Use WebP format with JPEG fallback
- Filename: keyword-based (e.g., "pure-crystal-palworld-farming.jpg")
- Compress to < 200KB
