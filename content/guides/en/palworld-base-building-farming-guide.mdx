# Palworld Base Building Guide: Farming & Agriculture Layout

Building an efficient farming and agriculture base is one of the most important milestones in Palworld. Without a dedicated food production base, your breeding operations will stall and your workers will be underfed and unproductive. This Palworld base building guide walks you through the complete setup for a farming base that can produce 20+ Cakes per day, plus surplus Salad, Pizza, and Jam-Filled Bread ingredients.

Whether you are setting up your second base or converting your main headquarters into a production hub, this guide covers location selection, building layout, Pal assignment, and breeding chains for agricultural Pals.

## Why Agriculture Bases Matter

In Palworld, cooked food dramatically outperforms raw ingredients:

- **Cooked food restores more hunger** and some dishes recover SAN
- **Salad and Pizza** provide work speed buffs or slow hunger decay
- **Jam-Filled Bread** has a 60-minute spoil timer, ideal for carrying while exploring
- **Attack-boosting cooked dishes** are essential for endgame tower bosses

For breeding specifically, a base with 6+ breeding pairs can consume 20+ Cakes per day. Buying all ingredients from merchants becomes prohibitively expensive, especially in multiplayer where money exploits are patched out. A self-sufficient agriculture base is the only sustainable solution.

## Base Location Requirements

Your agriculture base needs specific conditions to run smoothly:

### Must-Have Criteria

| Criterion | Reason |
|-----------|--------|
| No base raids | Protect your infrastructure (disable raids if preferred) |
| Flat terrain | Prevents Pals from getting stuck on slopes |
| Minimal natural resources | Trees and rocks interfere with crop layouts |
| No logging/quarrying nearby | Mining and chopping interrupt farming workflows |

### Recommended Locations

- The **top-left island** on the main map is excellent: flat, no raids (low enemy count), close to Volcano and Snow biomes, and has Iron Ore nodes
- Any large flat plain away from raid zones works if you cover the ground with foundations

## Building Layout

### Essential Structures

| Structure | Quantity | Notes |
|-----------|----------|-------|
| Pal Terminal | 1 | Center of base |
| Feed Box | 1+ | Store Pizza for work speed buff |
| Wheat Plantation | 7+ | Highest volume crop; grows slowest |
| Red Berry Plantation | 4-5 | Secondary crop for Cake |
| Tomato Plantation | 1 | For Salad production |
| Lettuce Plantation | 1 | For Salad production |
| Flour Mill | 1-2 | Convert Wheat → Flour; leave clearance around it |
| Ranch | 2+ | 4 Pals max per Ranch; for Eggs, Milk, Honey |
| Fountain | 1 | Work speed buff |
| Flower Bed | 1 | Work speed buff |
| Granary | 1 | Work speed buff |
| Chest/Wooden Chest | Several | Overflow storage for agricultural products |
| Holy Altar | 1 | If producing cooked food on-site |

### Layout Principles

1. **Foundations first**: Cover all ground with foundations to prevent natural resource respawn
2. **Straight-line paths**: Place rest areas, feeding areas, and work areas in direct lines with no obstacles
3. **No beds or hot springs inside the farm**: Pals detour to use them, breaking workflow
4. **Mill clearance**: Leave extra space around the Flour Mill, especially if using Sewefia for watering
5. **Ranch separation**: Keep Ranches away from crop fields to prevent Egg/Milk carrying from interrupting farming

## Optimal Pal Assignments

### Tier 1: Must-Have Agricultural Pals

| Pal | Role | Why |
|-----|------|-----|
| **Lylepanda** (百合女王) | Planting + Harvesting | Best combined farming ability |
| **Sewefia/Relaxaurus** (覆海龙) | Watering | Top-tier watering skill |
| **Penking** (皮皮鸡) | Egg production | Ranch work; upgrade to 4-star |
| **Reindox** (波霸牛) | Milk production | Ranch work; 3-4 star ideal |
| **Beeguard** (骑士蜂) | Honey production | Ranch work; upgrade stars for yield |

### Tier 2: Supporting Pals

| Pal | Role | Notes |
|-----|------|-------|
| **Katress** (唤夜兽) | Night harvesting | Extends harvest window for ~30% more output |
| **Pyromond** (狱阎王) | Fire/Cooking | 3rd-tier fire for advanced food production |
| **Anubis** | Crafting/Mining | Useful for byproduct processing |

### Critical Rule: No Carrying Pals in Crop Fields

Pals with the Transport work suitability will carry Eggs and Milk from Ranches instead of farming. This destroys crop productivity. Assign Pals without Transport to your crop fields.

### Work Passive Priority

For all agricultural Pals, prioritize these passives (in order):
1. **Craftsmanship** (工匠精神) — +50% work speed
2. **Serious** (认真) — +50% work speed
3. **Workaholic** (社畜) — +50% work speed
4. **Rare** (稀有) — +50% work speed

The "Holy Worker" combination of all four yields the fastest possible agricultural output.

## Agricultural Pal Breeding Chains

### Method A: Cattiva Base (5 Generations)

This chain uses only common starting-area Pals and produces top-tier farming Pals:

```
1. Breed 4-golden Cattiva (棉悠悠)
2. Cattiva + Dumud (企丸王) = Springit (踏春兔)
3. Springit + Dazzi (滑水蛇) = Gootrax (花丽娜)
4. Gootrax + Grizzbolt (叶胖达) = Lylepanda (百合女王)
5. Lylepanda + Frostallion/Foxparks/Horus = Relaxaurus/Sewefia (覆海龙)
```

All white (no-passive) intermediate Pals can be caught in the wild.

### Method B: Flopie Base (6 Generations, Faster Derivatives)

This chain produces more useful byproducts (Anubis, Level 3 fire):

```
1. Breed 4-golden Flopie (幻悦蝶)
2. Flopie + Flame Breeze (烽歌龙) = Anubis
3. Anubis + Flame Breeze = Horus
4. Flopie + Pengullet (猎狼) = Gootrax (花丽娜)
5. Gootrax + Green Panda (绿胖达) = Lylepanda (百合女王)
6. Horus + Lylepanda = Relaxaurus (覆海龙)
```

### Which Method to Choose?

- **Cattiva method**: Faster to 4-golden farming Pals (5 generations); Cattiva is extremely common, increasing shiny odds
- **Flopie method**: Produces Anubis and Level 3 fire Horus along the way; better overall value but harder to get 4-golden Flopie

### Scaling Tips

- If Wheat and Berries are overflowing, reduce crop slots and add more Ranch Pals
- Upgrade Beeguard stars for better Honey yield
- Run 2 Ranches (8 Pals total) if you need more animal products
- Visit your base every 1-2 days to collect products

## Tips from Experienced Players

Based on community-tested strategies:

1. **Crouch/roll cancels animation recovery** — including shooting, melee, and reloading. Press C in rhythm while mining for ~1.5× speed.
2. **Resources are base-global** — place a Wooden Chest next to every resource node so Pals deposit materials immediately instead of carrying them across the map.
3. **Mass resource collection trick** — build a production building, click "MAX craft all" then cancel. All resources across the base converge into that building. Use this near your Pal Terminal for efficient moving.
4. **Overencumbered? Use the grappling gun** — teleport yourself and your heavy cargo instantly.
5. **Disable death penalty** in custom settings — you can suicide (ESC) to teleport back to base anywhere on the map, great for moving超重 cargo.

## FAQ

### Do I need a kitchen in my farming base?

Not necessarily. Cooking requires a fire Pal and possibly a generator Pal for the Electric Kitchen. If you are short on Pals, send raw ingredients to your main base for cooking instead.

### How many Ranches do I need?

Each Ranch holds 4 Pals. For 20 Cakes/day, you need roughly 2 Ranches (4-star Penking ×2, 3-star Reindox ×1, 1-2 star Beeguard ×1, plus one flexible slot).

### Can I use the console to boost production?

The console has limited impact. Agriculture crop growth time is not affected by any multiplier. Livestock yield only scales with Pal star level, not work speed passives. The only useful console adjustment is increasing nighttime speed for more breeding time.
