# 幻兽帕鲁服务器搭建攻略：3分钟开启多人联机

独自游玩幻兽帕鲁很有趣，但搭建自己的多人服务器能彻底改变游戏体验。运行专用服务器意味着即使你关闭游戏，你的世界也会持续运行——你的好友随时可以加入，你可以完全控制每个游戏设置——经验倍率、死亡惩罚、帕鲁刷新率等等。本攻略将引导你完成整个流程，从选择云服务商到邀请第一位好友，只需不到三分钟的实际配置时间。

无论你是零服务器经验的完全新手，还是寻找简化工作流的资深管理员，本攻略都能涵盖你顺畅运行幻兽帕鲁服务器所需的一切。

## 为什么你需要专用服务器

在开始搭建之前，值得了解为什么专用服务器比其他方案更好：

- **持久世界** — 即使主机玩家离线，游戏世界也会持续运行。你的帕鲁继续工作，你的基地继续产出，时间继续流逝。
- **无主机依赖** — 在监听服务器模式下（点击"主持"时的默认模式），主机关闭游戏时服务器就会关闭。专用服务器独立于任何玩家运行。
- **完全配置控制** — 通过配置文件调整经验倍率、死亡惩罚、帕鲁刷新率、昼夜循环速度以及数十种其他设置。
- **更好的性能** — 专用云服务器比在家庭电脑上运行服务器有更稳定的CPU和内存分配，特别是4人以上组队时。
- **全天候可用** — 你的朋友可以随时登录，不受你的时间安排限制。这对不同时区的玩家至关重要。

## 服务器配置要求

幻兽帕鲁服务器比你想象的更吃资源。以下是最低和推荐配置：

| 资源 | 最低要求 | 推荐配置 |
|------|----------|----------|
| CPU | 4核 | 4核以上（高单线程性能） |
| 内存 | 16 GB | 32 GB（8人以上且有大型基地） |
| 存储 | 20 GB SSD | 40+ GB NVMe SSD |
| 带宽 | 10 Mbps | 50+ Mbps |
| 系统 | Ubuntu 22.04 / Windows Server 2022 | Ubuntu 22.04 LTS |

**重要提示：** Palworld服务器进程（PalServer）是单线程应用。时钟频率比核心数更重要。选择基础CPU频率高（3.0 GHz+）的云实例。

> **专业提示：** 不要尝试在2核/4GB的实例上运行幻兽帕鲁服务器。一旦两个或更多玩家开始在彼此附近建造基地，它就会在负载下崩溃。16GB内存是硬性最低要求。

## 云服务商选项与价格对比

多家云服务商提供适合幻兽帕鲁服务器的实例。以下是最热门选项的对比：

### 阿里云 (Alibaba Cloud)

阿里云为幻兽帕鲁服务器提供了出色的性价比，特别是对亚太地区的玩家。他们的ECS（弹性计算服务）4核16GB实例通常月费约200-350元，取决于地域和付费方式。主要优势包括亚洲玩家低延迟、灵活的计费方式（按量付费或包年包月），以及易用的控制台和一键服务器模板。

### 腾讯云

腾讯云同样是国内玩家的优秀选择，4核16GB轻量应用服务器月费约150-300元。优势在于国内网络质量极佳、操作简单、有专门的游戏服务器解决方案。

### AWS (Amazon Web Services)

AWS在全球提供可靠的基础设施。合适的EC2实例（t3.xlarge或m5.xlarge）按需约$50-80/月，预留实例更便宜。AWS在网络可靠性和全球区域覆盖方面表现出色，适合国际分布的好友群组。

### 其他选项

- **Vultr** — 性价比高，16GB内存实例起价约$24/月。适合北美和欧洲。
- **DigitalOcean** — 与Vultr价格相近，界面更简洁。适合新手。
- **Hetzner** — 欧洲服务商，性价比极高。最适合欧洲玩家。

对于大多数玩家，阿里云或AWS提供了价格、性能和易用性的最佳平衡。

## 三步搭建服务器

### 第一步：创建并连接云服务器

在你选择的云服务商注册并创建新实例，配置如下：

1. 选择 **Ubuntu 22.04 LTS** 作为操作系统
2. 选择至少 **4核16GB内存** 的实例类型
3. 分配至少 **40GB SSD存储**
4. 配置防火墙/安全组，允许入站流量通过 **8211 (UDP)** 和 **27015 (UDP)** 端口
5. 记录你服务器的公网IP地址

通过SSH连接到服务器：

```bash
ssh root@你的服务器IP
```

### 第二步：安装幻兽帕鲁服务端

连接后，运行以下命令安装SteamCMD并下载幻兽帕鲁专用服务器：

```bash
# 更新系统包
sudo apt update && sudo apt upgrade -y

# 安装依赖
sudo apt install -y lib32gcc-s1 lib32stdc++6 tmux curl

# 创建专用用户（不要用root运行）
sudo useradd -m -s /bin/bash palworld
sudo su - palworld

# 下载安装SteamCMD
mkdir ~/steamcmd && cd ~/steamcmd
curl -sSqL https://steamcdn-a.akamaihd.net/client/installer/steamcmd_linux.tar.gz | tar xzf -

# 安装幻兽帕鲁专用服务器
./steamcmd +force_install_dir ~/palworld-server +login anonymous +app_update 2394010 validate +quit
```

此过程下载约5-8GB的服务器文件。等待安装完成。

### 第三步：配置并启动服务器

创建启动脚本以便轻松启动：

```bash
cd ~/palworld-server
nano start-server.sh
```

添加以下内容：

```bash
#!/bin/bash
./PalServer.sh -useperfthreads -NoAsyncLoadingThread -UseMultithreadForDS -port=8211
```

赋予执行权限并启动：

```bash
chmod +x start-server.sh
./start-server.sh
```

要持久运行（断开SSH后继续运行），使用tmux：

```bash
tmux new -s palworld
./start-server.sh
# 按Ctrl+B然后D来分离会话
```

你的服务器现在已在8211端口上线！

## 游戏配置选项

服务器配置文件位于：

```
~/palworld-server/Pal/Saved/Config/LinuxServer/PalWorldSettings.ini
```

用文本编辑器打开，修改 `OptionSettings` 括号之间的设置。以下是最有影响的选项：

### 经验与进度

- **ExpRate** — 控制经验获取倍率。默认1.0。设为2.0为双倍经验，0.5为更慢更肝的体验。
- **DropItemRate** — 调整敌人和资源节点的掉落数量。更高值意味着更多战利品。
- **CollectionDropRate** — 控制从树木、岩石和矿石节点采集的资源数量。

### 死亡与惩罚

- **DeathPenalty** — 控制死亡时的损失。选项包括：
  - `None` — 无任何惩罚
  - `DropItem` — 掉落携带物品但保留装备
  - `DropEquipment` — 掉落物品和装备（默认）
  - `DropEverything` — 失去一切包括帕鲁背包物品
- **AutoResetWorrierHpOnDead** — 玩家复活时自动治疗帕鲁

### 世界与玩法

- **DayTimeSpeedRate** 和 **NightTimeSpeedRate** — 调整白天和夜晚的时间流逝速度。更高值使白天/黑夜更短。
- **PalSpawnNumRate** — 倍增野生帕鲁刷新密度。更高值意味着世界中有更多野生帕鲁。
- **PalStomachDecreaceRate** — 帕鲁饥饿速度。更低值意味着更少的食物消耗。
- **PlayerAutoHPRegeneRate** — 玩家生命恢复速度。

### 服务器身份

- **ServerName** — 你的服务器公开名称（在服务器浏览器中显示）
- **ServerDescription** — 服务器浏览器的简短描述
- **AdminPassword** — 游戏内管理员命令密码
- **ServerPassword** — 留空为公开服务器，或设置密码为私人会话

## 端口配置

幻兽帕鲁需要两个UDP端口：

| 端口 | 协议 | 用途 |
|------|------|------|
| 8211 | UDP | 主游戏流量 |
| 27015 | UDP | 服务器查询和Steam匹配 |

确保你的云服务商防火墙（安全组）允许这两个端口的入站UDP流量。在阿里云上，这在ECS控制台的安全组中配置。在AWS上，在EC2安全组设置中配置。

如果玩家无法连接，首先验证防火墙规则——这是最常见的连接问题。

## 连接到你的服务器

### 方法一：直接连接（最可靠）

1. 启动幻兽帕鲁
2. 点击 **加入多人游戏**
3. 在底部输入你服务器的IP和端口：`你的服务器IP:8211`
4. 点击 **连接**

### 方法二：服务器浏览器

1. 启动幻兽帕鲁
2. 点击 **加入多人游戏**
3. 如果你配置了ServerName，你的服务器应该出现在列表中
4. 点击 **连接**

> **注意：** 服务器浏览器列表可能需要最多5分钟才能在服务器启动后出现。如果没有显示，请改用直接连接。

## 服务器管理技巧

### 重启服务器

要干净地重启服务器：

```bash
tmux attach -t palworld
# 按Ctrl+C停止服务器
./start-server.sh  # 重新启动
```

定期重启（每天一次）有助于防止内存泄漏并保持性能稳定。

### 备份你的世界

存档数据存储在：

```
~/palworld-server/Pal/Saved/SaveGames/
```

定期备份。一个简单的cron任务可以自动化每日备份：

```bash
# 添加到crontab (crontab -e)
0 3 * * * tar -czf ~/backups/palworld-$(date +\%Y\%m\%d).tar.gz ~/palworld-server/Pal/Saved/SaveGames/
```

### 更新服务器

当幻兽帕鲁发布新版本时：

```bash
tmux attach -t palworld
# Ctrl+C停止
cd ~/steamcmd
./steamcmd +force_install_dir ~/palworld-server +login anonymous +app_update 2394010 validate +quit
./start-server.sh
```

### 监控性能

使用 `htop` 监控CPU和内存使用。如果在16GB实例上持续看到内存使用超过14GB，考虑升级或在配置文件中减少基地数量限制。

## 常见问题

### 我能在家里电脑上运行幻兽帕鲁服务器吗？

可以，但不推荐长期游玩。家庭网络通常上行带宽有限、IP动态变化，而且服务器只在你电脑开机时运行。云服务器解决了所有这些问题。

### 运行幻兽帕鲁服务器要花多少钱？

阿里云或同等服务商通常每月200-350元。具体费用取决于实例大小、地域和计费方式。测试用按量付费最便宜；长期运行包月更划算。

### 服务器能支持多少玩家？

4核/16GB服务器舒适支持4-8名玩家。12-16名玩家需要升级到32GB内存。超过16名玩家，需要专门的游戏服务器提供商提供的优化幻兽帕鲁主机。

### 为什么我的朋友连不上我的服务器？

按顺序检查三点：（1）防火墙/安全组是否允许8211和27015端口的入站UDP流量？（2）服务器是否真的在运行？（3）你和朋友是否是相同游戏版本？版本不匹配是第二大常见原因。

### 我能把单人存档转移到专用服务器吗？

可以。将存档文件从 `%LOCALAPPDATA%\Pal\Saved\SaveGames`（Windows）复制到服务器的 `Pal/Saved/SaveGames/` 目录。确保玩家Steam ID文件夹结构匹配。

### 最佳服务器位置是哪里？

选择离大多数玩家最近的区域。延迟在幻兽帕鲁中很重要，特别是在战斗和建造时。如果你的群组分散在不同区域，选择中间点。

### 如何在自己的服务器上当管理员？

将你的Steam ID添加到服务器配置目录的 `AdminList.txt` 文件中。你也可以在PalWorldSettings.ini中设置 `AdminPassword`，并在游戏内使用管理员控制台进行身份验证。

服务器常见问题排查，请查看我们的[幻兽帕鲁服务器报错排查攻略](/zh/guides/palworld-server-troubleshooting-errors-guide)。
