# 幻兽帕鲁基地建设攻略：农业种植布局

建造一个高效的农业种植基地是幻兽帕鲁中最重要的里程碑之一。没有专门的食品生产基地，你的配种运作就会停滞，工人帕鲁也会因食物不足而效率低下。本攻略将带你完整搭建一个日产20+蛋糕的农业基地，还能额外产出沙拉、披萨和果酱面包的原料。

无论你是要建设第二基地，还是将主基地改造成生产中心，本攻略都涵盖选址、建筑布局、帕鲁分配和农业帕鲁配种链。

## 为什么农业基地如此重要

在幻兽帕鲁中，烹饪食物远超生食材的效果：

- **烹饪食物恢复更多饥饿度**，部分菜品还能恢复SAN值
- **沙拉和披萨**提供工作速度加成或减缓饥饿度下降
- **果酱面包**有60分钟保质期，适合外出探索时携带
- **攻击加成的烹饪菜品**是后期塔Boss战的必需品

具体到配种方面，一个拥有6+对配种帕鲁的基地每天可能消耗20+个蛋糕。从商人处购买所有食材成本极高，尤其是在多人服务器中金钱 exploit 已被修复的情况下。一个自给自足的农业基地是唯一可持续的解决方案。

## 基地选址要求

你的农业基地需要满足特定条件才能顺畅运转：

### 必要条件

| 条件 | 原因 |
|------|------|
| 无基地袭击 | 保护基础设施（如偏好可关闭袭击） |
| 平坦地形 | 防止帕鲁在斜坡上卡住 |
| 最少自然资源 | 树木和岩石会干扰农作物布局 |
| 附近无伐木/采石场 | 采矿和砍伐会打断农业工作流 |

### 推荐位置

- 主地图**左上角岛屿**极佳：平坦、无袭击（敌人数量少）、靠近火山和雪地生物群系，且有铁矿石节点
- 任何远离袭击区的大型平坦空地都可以，只要用地板覆盖地面即可

## 建筑布局

### 必要建筑

| 建筑 | 数量 | 备注 |
|------|------|------|
| 帕鲁终端 | 1 | 基地中心 |
| 饲料箱 | 1+ | 存放披萨以获取工作速度加成 |
| 小麦种植园 | 7+ | 最大量作物；生长最慢 |
| 红色浆果种植园 | 4-5 | 蛋糕的次要作物 |
| 番茄种植园 | 1 | 用于生产沙拉 |
| 生菜种植园 | 1 | 用于生产沙拉 |
| 磨粉机 | 1-2 | 将小麦 → 面粉；周围留足空间 |
| 牧场 | 2+ | 每个牧场最多4只帕鲁；用于产蛋、奶、蜂蜜 |
| 喷泉 | 1 | 工作速度加成 |
| 花坛 | 1 | 工作速度加成 |
| 粮仓 | 1 | 工作速度加成 |
| 宝箱/木箱 | 若干 | 农产品溢出存储 |
| 神圣祭坛 | 1 | 如在基地内烹饪则需要 |

### 布局原则

1. **先铺地板**：用地板覆盖全部地面，防止自然资源重生
2. **直线路径**：休息区、进食区和工作区呈直线排列，中间无障碍物
3. **农场内不放床和温泉**：帕鲁会绕路去使用，打断工作流
4. **磨粉机留空**：磨粉机周围留足额外空间，尤其是使用覆海龙 (Sewefia) 浇水时
5. **牧场隔离**：牧场远离农作物田地，防止搬运蛋/奶打断种植工作

## 最佳帕鲁分配

### 第一梯队：必备农业帕鲁

| 帕鲁 | 职责 | 原因 |
|------|------|------|
| **百合女王 (Lylepanda)** | 种植 + 收获 | 最佳综合农业能力 |
| **覆海龙 (Sewefia/Relaxaurus)** | 浇水 | 顶级浇水技能 |
| **皮皮鸡 (Penking)** | 产蛋 | 牧场工作；建议升至4星 |
| **波霸牛 (Reindox)** | 产奶 | 牧场工作；3-4星为佳 |
| **骑士蜂 (Beeguard)** | 产蜂蜜 | 牧场工作；升星提升产量 |

### 第二梯队：辅助帕鲁

| 帕鲁 | 职责 | 备注 |
|------|------|------|
| **唤夜兽 (Katress)** | 夜间收获 | 延长收获时间窗，产量提升约30% |
| **狱阎王 (Pyromond)** | 火焰/烹饪 | 3阶火焰，用于高级食品生产 |
| **阿努比斯 (Anubis)** | 手工/采矿 | 用于副产物加工 |

### 关键规则：农田中不要分配搬运帕鲁

拥有"搬运"工作适性的帕鲁会从牧场搬运鸡蛋和牛奶，而不是种田。这会严重降低农作物产出。请在农田中分配没有搬运能力的帕鲁。

### 工作被动优先级

所有农业帕鲁按以下顺序优先选择被动（优先级从高到低）：
1. **工匠精神 (Craftsmanship)** — 工作速度 +50%
2. **认真 (Serious)** — 工作速度 +50%
3. **社畜 (Workaholic)** — 工作速度 +50%
4. **稀有 (Rare)** — 工作速度 +50%

四种被动全有的"圣工"组合可产生最快的农业产出。

## 农业帕鲁配种链

### 方案A：棉悠悠起步（5代）

此路线仅使用常见的起始区域帕鲁，就能培育出顶级农业帕鲁：

```
1. 培育4金棉悠悠 (Cattiva)
2. 棉悠悠 + 企丸王 (Dumud) = 踏春兔 (Springit)
3. 踏春兔 + 滑水蛇 (Dazzi) = 花丽娜 (Gootrax)
4. 花丽娜 + 叶胖达 (Grizzbolt) = 百合女王 (Lylepanda)
5. 百合女王 + 霜牙王/狐帕/荷鲁斯 = 覆海龙 (Relaxaurus/Sewefia)
```

所有无被动的中间产物可在野外直接捕捉白色个体。

### 方案B：幻悦蝶起步（6代，更多副产物）

此路线能同时产出更多有用的副产物（阿努比斯、3阶火焰）：

```
1. 培育4金幻悦蝶 (Flopie)
2. 幻悦蝶 + 烽歌龙 (Flame Breeze) = 阿努比斯 (Anubis)
3. 阿努比斯 + 烽歌龙 = 荷鲁斯 (Horus)
4. 幻悦蝶 + 猎狼 (Pengullet) = 花丽娜 (Gootrax)
5. 花丽娜 + 绿胖达 (Green Panda) = 百合女王 (Lylepanda)
6. 荷鲁斯 + 百合女王 = 覆海龙 (Relaxaurus)
```

### 如何选择？

- **棉悠悠路线**：更快获得4金农业帕鲁（5代）；棉悠悠极为常见，提升闪光概率
- **幻悦蝶路线**：沿途产出阿努比斯和3阶火焰荷鲁斯；整体价值更高但4金幻悦蝶更难获得

## 产出基准

在完全优化的农业基地中（4×百合女王 + 4×覆海龙，以披萨为帕鲁食物）：

| 资源 | 日产出 |
|------|--------|
| 小麦 | 800+ |
| 红色浆果 | 数百 |
| 生菜 | 3+次收获/天（有夜间收获者可达4次） |
| 面粉 | 100-200/天 |
| 鸡蛋 | 足够制作20个蛋糕 |
| 牛奶 | 足够制作20个蛋糕 |
| 蜂蜜 | 浮动（牧场产出有波动） |

### 扩展建议

- 如果小麦和浆果溢出，减少种植槽位，增加牧场帕鲁
- 提升骑士蜂星级以增加蜂蜜产量
- 如需更多畜产品可运行2个牧场（共8只帕鲁）
- 每1-2天访问一次基地收集产品

## 老玩家经验技巧

基于社区验证的策略：

1. **蹲伏/翻滚取消动作后摇** — 包括射击、近战和装弹。采矿时有节奏地按C可达到约1.5倍采矿速度。
2. **资源是基地全局共享的** — 在每个资源节点旁放一个木箱，帕鲁会立即存放材料，而不是搬运到地图另一头。
3. **大量资源收集技巧** — 建造一个生产建筑，点击"MAX制作全部"然后取消。基地所有资源会汇聚到该建筑。在帕鲁终端旁边使用此技巧方便搬运。
4. **负重超标？使用抓钩枪** — 瞬间传送到目标位置，包括你超载的货物。
5. **在自定义设置中关闭死亡惩罚** — 你可以自杀（ESC键）在地图任意位置传送回基地，非常适合搬运超重货物。

## 常见问题

### 农业基地需要厨房吗？

不一定。烹饪需要一只火焰帕鲁，如果要做电厨房还需要发电机帕鲁。如果帕鲁不够用，把原材料送到主基地烹饪。

### 需要多少个牧场？

每个牧场容纳4只帕鲁。要日产20个蛋糕，大约需要2个牧场（4星皮皮鸡 ×2、3星波霸牛 ×1、1-2星骑士蜂 ×1，再加一个灵活槽位）。

### 可以用控制台提升产出吗？

控制台效果有限。农业作物生长时间不受任何倍率影响。畜牧产量只随帕鲁星级缩放，与工作速度被动无关。唯一有用的控制台调整是加快夜间速度以增加更多配种时间。

## 总结

农业基地是幻兽帕鲁中任何认真经营的玩家的核心支柱。凭借正确的布局、帕鲁分配和配种链，你可以持续日产20+蛋糕，并供应其他基地所需的所有食物。先用棉悠悠路线快速见效，然后扩展到幻悦蝶路线以获取最大副产物价值。
