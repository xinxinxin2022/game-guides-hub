# 幻兽帕鲁帕鲁球策略攻略：何时使用哪种球

帕鲁球是幻兽帕鲁中最重要的消耗品资源。每次捕获尝试都要消耗帕鲁球，在错误的时间使用错误的球类型，可能意味着成功把传奇帕鲁收入囊中和白白浪费珍贵材料之间的天壤之别。本攻略涵盖所有帕鲁球类型、捕获率背后的数学原理，以及如何精确使用每种球以达到最高效率。

无论你是想抓到第一只普通帕鲁，还是挑战四大圣兽，理解帕鲁球策略都能在整个游戏过程中为你节省数百份资源。大部分玩家浪费的帕鲁球远超必要，原因就在于他们不理解捕获率系统。本攻略帮你解决这个问题。

## 帕鲁球基础

帕鲁球是可投掷的装置，用于捕获虚弱的帕鲁。当帕鲁的HP足够低时，对它投掷帕鲁球就会触发捕获判定。成功概率取决于三个因素：

1. **帕鲁剩余HP百分比** — HP越低，捕获率越高
2. **使用的帕鲁球类型** — 越高级的球基础捕获率越高
3. **帕鲁的捕获率等级** — 有些帕鲁天生就比其他帕鲁更难捕捉

游戏中的每只帕鲁都属于一个捕获率等级，该等级决定了它的捕捉难度，与帕鲁球类型无关。

## 所有帕鲁球类型详解

幻兽帕鲁共有五种不同的帕鲁球，各有不同的捕获率、制作成本和获取时机。以下是完整解析。

### 帕鲁球（基础）

| 属性 | 详情 |
|------|------|
| 捕获率倍率 | 1.0x（基准） |
| 解锁等级 | 1级（初始物品） |
| 制作成本 | 1个铜锭、1个石头 |
| 获取时机 | 游戏一开始 |
| 最佳用途 | 普通帕鲁、低等级野生帕鲁 |

基础帕鲁球是你的起始捕获装置。它对普通帕鲁和低等级敌人效果尚可，但面对越来越强的对手时会变得愈发力不从心。你应该在15级左右开始淘汰基础球。

### 高级球（Mega Sphere）

| 属性 | 详情 |
|------|------|
| 捕获率倍率 | 1.5x |
| 解锁等级 | 12级 |
| 制作成本 | 3个铜锭、2个石头、1个帕鲁液 |
| 获取时机 | 前期 |
| 最佳用途 | 中级野生帕鲁（10-25级） |

高级球是第一个升级档位。它比基础球提供50%的捕获率加成，对中级帕鲁显著更有效。制作成本在前期仍然可以承受，所以你可以比较大方地使用。

> **进阶提示：** 对于10-25级的帕鲁，高级球是性价比最高的选择。不要把超级球浪费在高级球就能稳定捕获的普通帕鲁上。

### 超级球（Ultra Sphere）

| 属性 | 详情 |
|------|------|
| 捕获率倍率 | 2.0x |
| 解锁等级 | 28级 |
| 制作成本 | 5个铜锭、3个石头、2个帕鲁液、1个纯净水晶 |
| 获取时机 | 中期 |
| 最佳用途 | 高级野生帕鲁（25-40级）、小型Boss |

超级球将基础捕获率翻倍。这是中期大部分时间的主力球种。它足够强大，能可靠地捕获大部分非Boss帕鲁，是你日常探索的首选球种。

### Hyper球（Hyper Sphere）

| 属性 | 详情 |
|------|------|
| 捕获率倍率 | 3.0x |
| 解锁等级 | 40级 |
| 制作成本 | 5个精炼金属、3个纯水晶体、2个帕鲁液 |
| 获取时机 | 中后期 |
| 最佳用途 | Boss、圣兽、传奇帕鲁 |

Hyper球将基础捕获率提升至三倍。制作成本高昂，应该留给高价值目标。把Hyper球用在普通帕鲁上是严重的资源浪费。

### 传奇球（Legendary Sphere）

| 属性 | 详情 |
|------|------|
| 捕获率倍率 | 5.0x |
| 解锁等级 | 50级 |
| 制作成本 | 10个精炼金属、5个纯水晶体、3个古代技术点 |
| 获取时机 | 后期 |
| 最佳用途 | 圣兽、顶级传奇Boss |

传奇球拥有高达5倍的捕获率倍率，但需要后期材料和古代技术点才能制作。应该只用在游戏中最稀有、最强大的帕鲁上。一局游戏下来你能制作的传奇球寥寥无几，务必珍惜使用。

## 捕获率公式

理解捕获率背后的数学原理，能让你做出最优的球种选择和投掷时机判断。

### 基础公式

简化的捕获率公式如下：

```
捕获率 = 基础捕获率 × 帕鲁球倍率 × HP系数 × 状态系数

其中：
- 基础捕获率 = 帕鲁的固有可捕获性（因种类而异）
- 帕鲁球倍率 = 1.0x 到 5.0x，取决于帕鲁球类型
- HP系数 = (1 - 当前HP百分比 × 0.5)
    100% HP时：0.5
    50% HP时：0.75
    25% HP时：0.875
    10% HP时：0.95
    1% HP时：0.995
- 状态系数 = 1.0（正常）、1.25（燃烧/冻结）、1.5（睡眠）
```

### 不同HP百分比的捕获率

以下是使用基础帕鲁球时，HP百分比对捕获率的影响：

| 剩余HP | HP系数 | 有效捕获率 |
|--------|--------|------------|
| 100% | 0.50 | 基础值的0.50倍 |
| 75% | 0.625 | 基础值的0.625倍 |
| 50% | 0.75 | 基础值的0.75倍 |
| 25% | 0.875 | 基础值的0.875倍 |
| 10% | 0.95 | 基础值的0.95倍 |
| 1% | 0.995 | 基础值的0.995倍 |

> **进阶提示：** HP系数在25%以下增长明显放缓。把帕鲁从10%打到1%只多获得4.5%的捕获率，但从50%打到25%能获得12.5%。最佳投掷窗口是15-30%HP——而不是1%。

### 状态效果与捕获率

状态效果提供有意义的捕获率加成：

| 状态效果 | 捕获率加成 | 持续时间 | 施加方式 |
|----------|-----------|----------|----------|
| 燃烧 | +25% | 数秒 | 火系攻击 |
| 冻结 | +25% | 数秒 | 冰系攻击 |
| 睡眠 | +50% | 直到被打醒 | 催眠攻击或道具 |
| 中毒 | +10% | 较长 | 毒系攻击 |
| 眩晕 | +15% | 短暂 | 雷系攻击 |

**睡眠是捕捉时最有价值的状态。** +50%的加成巨大，而且睡着的帕鲁无法闪避或反击，帕鲁球很容易命中。

## 何时使用哪种帕鲁球

以下是根据目标帕鲁等级和实际情况，何时使用每种帕鲁球的实用指南。

### 基础帕鲁球 — 1至15级目标

基础球适用于：
- 起始区域的普通帕鲁（捣蛋猫、草莽猪、雷灵鼠）
- 10级以下的帕鲁
- 用普通种类填满帕鲁图鉴
- 练习投掷以熟悉抛物线

**何时停用：** 12级解锁高级球后立即切换。

### 高级球 — 10至25级目标

高级球适用于：
- 10-25级的野生帕鲁
- 不想过度削弱就想捕获的帕鲁
- 前期阵容建设

**何时停用：** 28级解锁超级球后，或目标帕鲁超过25级时。

### 超级球 — 25至40级目标

超级球适用于：
- 25-40级的中期野生帕鲁
- 小Boss和首领帕鲁（削弱后）
- 中期阵容所需的帕鲁
- 日常探索和捕捉

**何时停用：** 目标为Boss级或传奇帕鲁时，需要Hyper球才能获得合理的捕获率。

### Hyper球 — 40级以上和Boss目标

Hyper球适用于：
- Boss帕鲁（野外Boss和地下城Boss）
- 圣兽（削弱至低HP后）
- 任何有传奇掉落或稀有刷新的帕鲁
- 必须抓到、绝不能失败的帕鲁

**不要用于：** 普通帕鲁、常规野生刷新、或30级以下的任何目标。

### 传奇球 — 圣兽和顶级传奇Boss

传奇球仅用于：
- 低HP的圣兽
- 顶级传奇Boss
- 最稀有的活动帕鲁

一局游戏你大概只能制作不到10个传奇球。每一个都应该用在最有价值的捕获目标上。

## 帕鲁球效率策略

除了知道该用哪种球，以下策略能最大化你的整体捕获效率。

### 策略一：HP窗口

正如捕获率公式所示，最佳投掷HP窗口是15-30%。低于15%时，每减少一点HP带来的捕获率提升微乎其微，却增加了意外击杀帕鲁的风险。高于30%时，捕获率显著下降。

**最佳投掷流程：**

1. 使用非致命攻击将目标削弱至约20%HP
2. 尽可能施加状态效果（睡眠 > 冻结/燃烧 > 中毒）
3. 投掷帕鲁球
4. 如果失败，将HP降至10%后再次投掷

### 策略二：状态叠加

投球之前一定要先施加状态效果。捕获率加成太重要了，不容忽视。

**最佳状态施加顺序：**

1. 使用催眠帕鲁或道具使目标入睡（+50%捕获率）
2. 如果没有催眠手段，使用冻结或燃烧（+25%捕获率）
3. 在状态效果持续期间投掷帕鲁球
4. 如果失败，重新施加状态后再投

> **进阶提示：** 捕获任务时带上一只专职催眠帕鲁。+50%的捕获率加成相当于免费送你一个球种升级。一颗对着睡眠目标投出的Hyper球，在正常HP下的捕获率等同于传奇球。

### 策略三：帕鲁球经济管理

仔细管理你的帕鲁球资源：

- **批量制作** — 一次制作20-30个当前档位的帕鲁球
- **区域过渡前备货** — 进入新区域前，确保有50个以上对应档位的帕鲁球
- **不要过度囤积** — 帕鲁球是用来用的，囤着不用不会让你的团队变强
- **优先分配纯水晶体** — 高阶帕鲁球需要纯水晶体，这是游戏中最稀有的制作材料。把它花在Hyper球和传奇球上，而不是低阶球

## 特殊捕获场景

有些帕鲁需要超出标准方法的特定策略。

### 捕获飞行帕鲁

飞行帕鲁因为空中位置的关系，更难被帕鲁球命中。

- **先把它打下来** — 使用冰系攻击冻结它在空中，使其坠落
- **利用高地** — 从悬崖或高建筑投掷，获得更好的角度
- **等待降落** — 大部分飞行帕鲁会定期着陆。在着陆动画时投掷

### 捕获夜行帕鲁

有些帕鲁只在夜间刷新，行为模式也不同。

- **携带光源** — 夜间视野降低
- **利用催眠策略** — 夜行帕鲁在其活跃时段通常更容易被催眠
- **备足超级球** — 夜行帕鲁往往等级较高

### 捕获群体目标

当帕鲁以群体形式刷新时，捕获其中一只可能会吓散其他成员。

- **孤立目标** — 先把一只帕鲁从群体中引出来再动手
- **使用范围催眠** — 催眠范围攻击能让整个群体入睡，让你在没有其他帕鲁干扰的情况下捕捉目标
- **先消灭护卫** — 有时候先把护卫全部击倒再专心捕捉目标更干脆

## 常见问题

### 幻兽帕鲁中最好用的帕鲁球是哪个？

最佳帕鲁球取决于你的目标。对于25级以下的普通帕鲁，高级球性价比最高。对于中期目标（25-40级），超级球在捕获率和成本之间达到最佳平衡。对于Boss、圣兽和传奇帕鲁，Hyper球必不可少。传奇球则留给游戏中最稀有、最困难的捕获目标。

### 幻兽帕鲁的捕获率公式怎么运作？

捕获率计算方式为：基础捕获率 × 帕鲁球倍率 × HP系数 × 状态系数。HP系数在满HP时为0.5，在1%HP时为0.995。状态效果会提供加成：睡眠+50%，冻结/燃烧+25%，中毒+10%。最佳投掷窗口是施加状态效果后在15-30%HP时投掷。

### 什么时候应该停止使用高级球？

当你在28级解锁超级球，或开始捕捉25级以上帕鲁时，就应该停止使用高级球。高级球在10-25级区间性价比最高，但对更高级的目标会变得不可靠。尽快过渡到超级球。

### 捕获传奇帕鲁最高效的方法是什么？

最高效的方法是：将传奇帕鲁削弱到15-20%HP，施加睡眠或冻结状态，然后投掷Hyper球或传奇球。睡眠状态提供+50%的捕获率加成，能大幅减少所需的帕鲁球数量。猎杀圣兽时务必携带至少20个Hyper球。

### 不削弱帕鲁能直接抓到吗？

理论上可以，但极其浪费。满HP的帕鲁只有其潜在捕获率的50%。在满HP时捕捉消耗的帕鲁球是在20%HP时捕捉的3-5倍。投球前务必将目标削弱到15-30%HP窗口。

### 如何获取制作高阶帕鲁球的纯水晶体？

纯水晶体通过在高海拔和洞穴地点开采水晶矿点获得。最高效的刷取地点是费利斯山顶（8-12个矿点）、水晶火山口洞穴（5-8个矿点）和冰封高地（4-6个矿点）。使用采矿帕鲁如白绒雪狐，搭配满级镐头和采矿食物增益，每小时可以产出60-80个纯水晶体。
